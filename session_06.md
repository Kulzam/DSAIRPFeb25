# Python Data Types
## Numeric Type
### Integer
    which stores integer data types (eg: 2 , 3)
    
### Float
    stores floating point numbers(eg:2.0)
### Complex
    c= a+jb  
## Text Type
### String
    strings are the values which we write inside "" or ''
    and in python input from the user is a string in default
## Sequence Type
### List
    it is a collection of elements which can either strings or 
    integers or floating point numbers
    it is Mutable( can insert or append or replace or delete values 
    from list ) and unordered  []
### Tuple
    tuple is collection of values which is immutable and ordered  ()
### Range
    it can give a sequence of nos from 0 default by incrementing one 
    to N-1  ( range(5)  it gives 0 to 4  , range(0,101,5) from zero 
    to 99 by incrementing 5)

## Mapping Type
### Dictionary 
    it is collection of unordered , mutable collection that stores 
    the data in key-value pairs . Each key is unique . if 
    accessing only keys - data.keys()
    accesssing only values- data.values()    {}
## Set Type
    it is used to store several items in a single variable .
    which has no duplicate values and unordered and unindexed . {}
    

## Boolean
    The boolean data type represent truth values , either TRUE or 
    FALSE , often used in conditional statements and expressions

## None Type
    the none data type represents the absence of a value or null value
